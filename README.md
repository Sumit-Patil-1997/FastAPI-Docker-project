# 🚀 FastAPI  Project 

A simple FastAPI authentication project with:

1. ✅ Basic FastAPI endpoint  
2. 🔐 JWT-based login & registration  
3. 🗃️ SQLite for managing users  
4. 🔑 Passwords hashed using SHA256  
5. 🐳 Docker + Docker Compose support
